# RNG Loot Simulator

An addictive **GUI-based RNG game** where players open different types of loot containers to obtain valuable items. The game focuses on chance, progression, and collecting rare rewards.

## Overview

RNG Loot Simulator is built around a simple but engaging gameplay loop:
Players open **Card Packs, Treasure Chests, and Gun Cases** to receive randomly generated items with different rarity levels. These items can be sold for in-game currency, allowing players to buy better loot containers with higher chances of rare rewards.

The goal is to build wealth, unlock stronger RNG options, and collect the rarest items in the game.

## Features

* **Multiple Loot Sources**

  * Card Packs
  * Treasure Chests
  * Gun Cases

* **Rarity System**

  * Common
  * Rare
  * Epic
  * Legendary
  * Mythic

* **Inventory System**

  * Store collected items
  * View rarity and value
  * Manage your collection

* **Sell System**

  * Sell items for in-game currency
  * Earn money to buy better loot

* **Progression Loop**

  * Open containers
  * Collect rare items
  * Sell items for money
  * Purchase higher-tier RNG containers
  * Repeat to chase better rewards

* **GUI-Based Gameplay**

  * Clean and interactive menus
  * Opening animations
  * Rarity highlights
  * Reward reveal effects

## Gameplay Loop

1. Purchase a loot container (pack, chest, or case).
2. Open it to receive a random item.
3. Check the item's rarity and value.
4. Sell the item or keep it in your inventory.
5. Use the money to buy better RNG containers.
